RTN Scheduling

The rtn_scheduling package is a tool developed for batch scheduling using the Resource Task Network (RTN) formulation with the Pyomo modeling language. It offers features like optimal schedule visualization through Gantt charts, network topology insights, resource level tracking, and an intuitive input format, aimed at improving the efficiency of scheduling tasks.

Using Machine Learning Surrogates for Produced Water Networks

This paper explores the integration of data-driven approaches for optimizing energy systems, focusing on using machine learning and predictive analytics for better performance and efficiency.

The proposed methodologies aim to enhance the management of energy production and consumption, leveraging real-time data to improve decision-making processes and reduce energy waste. The approach combines advanced modeling with practical applications, highlighting its effectiveness through case studies and experimental validation.

Optimization model and algorithms for the Unit Commitment problem

This paper presents a solution to the unit commitment problem by employing a decomposition method combined with the tight model from EGRET software, addressing its combinatorial complexity. Benchmark tests against four case studies demonstrate significant computational speedups without compromising solution quality. (Under review at ESCAPE-35)
