rtn_scheduling
A pipeline tool for solving RTN scheduling problems with data visualization capabilities.
Developed to streamline RTN scheduling, this Python package allows users to input data intuitively and generate schedules efficiently. It also features a built-in visualization module, enhancing usability.

Estimation-of-RUL
Estimates residual useful life (RUL) using first-principle and data-driven models.
Combines first-principles approaches with time-series models like ARIMA, LSTM, and Fourier transforms to estimate RUL. It serves as a valuable resource for predictive maintenance and system reliability analysis.
